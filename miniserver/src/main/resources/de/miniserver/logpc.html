Einrichten von Mel!
auf dem PC

1. Installation

Lade das .jar herunter und packe es in einen Ordner (zb /home/ich/mel/).
Dann startest du ein Terminal und gehst in jenen Ordner: cd /home/ich/mel
Nun lässt du Java dsa Programm ausführen: java -jar fx-bundle-1.0.jar
Auf der Konsole sollten nun diverse Ausgaben kommen und nach einer Weile das Hauptfenster erscheinen.
Nach dem Start erzeugt Mel den Ordner mel.data und die Datei mel.settings.json.
Diese solltest du aufheben und kannst hier die Einstellungen mit dem Texteditor verändern, falls nötig.

2. Andere Geräte verbinden

Gehe auf Paarung. Dann sollte es ähnlich wie auf folgendem Bild aussehen. In der linken Liste siehst du alle anderen Mel-Instanzen in deinem Netzwerk. Damit Mellas miteinander reden können, muss der Port 9966 erreichbar sein. Dazu musst du möglichweise deine Firewall konfigurieren. Für funktionierende Kommunikation werden ebenfalls die Ports 8888 und 8889 benötigt.

Drücke auf Unternehme eleganten Annäherungsversuch um zu verbinden. Der nachfolgende Dialog erscheint. Links siehst du dein Zertifikat und rechts das der anderen Mel. Prüfe ob die andere Mel den selben Hash anzeigt wie auf deinem Bildschirm. In diesem Fall e76337cdb....

Drücke auf Accept, falls du der anderen Mel vertraust.
